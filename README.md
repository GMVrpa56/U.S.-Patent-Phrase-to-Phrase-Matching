# U.S. Patent Phrase to Phrase Matching

## Overview
This project performs phrase-to-phrase matching using TF-IDF techniques and Ridge Regression on U.S. patent data.

## Technologies Used
- Python
- NLP
- TF-IDF
- Ridge Regression
- HTML
- CSV

## Files Included
- HTML project/report
- Dataset CSV file

## Workflow
1. Data preprocessing
2. Text cleaning
3. TF-IDF vectorization
4. Phrase similarity matching
5. Result analysis

## Dataset
U.S. Patent Phrase Matching dataset.
